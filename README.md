# WILLIAM BONAC FITNESS TEMPLE CRM

A custom Gym CRM system built for **William Bonac Fitness Temple**, designed and developed by **Simon Addai Boakye**.  
This system supports both **offline (PWA)** and **online (Vercel deployment)** access.

---

## ✨ Features
- 🔐 **Secure Login** (Admin, Staff, Sales roles)
- 📂 **Client Management**
  - Import clients via CSV
  - Manual client registration
  - Membership types: Registration, Monthly, 3‑Month, 6‑Month, Yearly
  - Payment methods: MoMo or Bank Card
  - Automatic expiry reminders (3 days before expiry)
- 💧 **Water Sales Tracking**
  - Small bottle (3 GH₵)
  - Medium bottle (5 GH₵)
  - Large bottle (8 GH₵)
- 📦 **Inventory Management**
  - Add new items with prices
  - Track sales and stock
- 📊 **Revenue Dashboard**
  - Daily revenue calculation
  - Attendance tracking
- 📱 **Progressive Web App (PWA)**
  - Works offline and online
  - Installable on mobile and desktop
  - Branded WBFT icons (192px, 512px, Apple Touch)

---

## 🚀 Deployment
- Hosted on [Vercel](https://vercel.com)  
- GitHub → Vercel CI/CD pipeline for automatic updates

---

## 🛠️ Tech Stack
- HTML5, CSS3, JavaScript
- Service Worker + Manifest (PWA)
- GitHub Actions + Vercel Deployment
